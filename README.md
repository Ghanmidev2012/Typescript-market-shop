# Typescript-market-shop
basic simple market made on typescript.

# who it work
to make input into typescript we use `readline` is the `node.js`module.
# how to get
to install we use this command:
```
npm install --save-dev @types/node

```
to get this module import for typescript
```typescript
import * as readline from 'readline'
```
# Where is file  index.ts

to find file, go to `src` folder and you find `index.ts`

# Who to run
to run file `index.ts` type in your `terminal`:
```
npx tsc
```
```
node build/index.js
```
